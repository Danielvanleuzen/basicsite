HTML Tags and Attributes

What are HTML Tags and Attributes?

Tags are elements of the HTML document used to specify how the document should be displayed by the browser. In HTML, each tag has its own specific meaning, which is (in general) common across all different browsers.

Attributes are associated with each tag to further define the tags.The general syntax is as follows: <tag attribute 1 = "value" attribute 2 = "value" ...>
